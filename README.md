# CEMS Project

College Event Management System (CEMS) is a web-based platform developed to manage college events, registrations, OD requests, and approvals efficiently.

## Features
- Student Registration
- Event Management
- OD Approval System
- Department-wise Access
- Admin & Officer Login
- Real-time Status Tracking

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
